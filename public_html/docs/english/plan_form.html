Subscription Plan Attributes

Plan ID
Enter a product ID for this product. This should be short, perhaps one word, and must be unique. If left blank, or not unique, then an ID will be automatically created.
Duration Period
Select the duration type, either "Fixed" or a time period. If "Fixed" is selected, enter the expiration date in the field to the right. If a period of time is selected, enter the number of periods in the field provided. For example, you may select "Month" and enter "6" to create a 6-month subscription.
Expiration
If this subscription has a fixed expiration date, enter it here in 'YYYY-MM-DD' format.
Price
Enter the price for this subscription.
Subscription Group
Select the glFusion group to which subscribers will be automatically added. This group can then be granted additional privileges.
Short Description
Enter a short description for this product. This will be used in the product listing.
Description
Enter a description for this product.
Enabled?
Check this box to enable this product. The subscription product will not be available for purchase if it is not enabled.
Show in Block?
Check this box to have this product shown in a "Subscribe" block. This will only be shown to users who are not subscribed already, or whose subscription will be expiring within early_renewal days.
Taxable?
If this subscription product is subject to sales tax, check this box.
At Registration?
Check this box to have new users automatically subscribed. Enter a number of days for the Trial Period, or leave blank to create a full subscription.
Trial Days
If a trial subscription at registration is enabled, enter the number of days for the trial. The trial subscription expires normally after this period and the subscriber can purchase an extension.
Grace Period
Enter the number of days after expiration that the subscription will remain active. This is intended to offer a short period during which the subscriber may renew without loss of service. Enter "0" to have the subscription terminated as soon as it expires.
Early Renewal
Enter the maximum number of days before a subscription expires that it may be renewed. In order to avoid loss of service, this should be set to at least a few days prior to expiration.
Purchase Allowed By
Select the user group that is allowed to purchase this subscription. Typically this will be "Logged-In Users"
Upgrade From
If this product is an upgrade, or can be an upgrade from another product, select that product here.

If there is an "upgrade price", enter that in the space provided. If this amount exists, current subscribers to the existing product will be offered the upgrade price while non-subscribers are offered the regular price entered above.

Optionally, upgrading a subscription may extend the subscription period to the same date as if the product was purchased normally.

Updates Profile
If the Profile plugin is installed and enabled, subscriptions can update the Member Type and Expiration Date fields in the user's profile record. This data can then be displayed on membership lists.

You can specify that the profile data is left unchanged, only the buyer's account is updated, or the buyer and all child accounts are updated.

  • None - No accounts are updated.
  • Buyer Only - The buyer's profile information is updated.
  • Buyer and Children - The buyer's profile information is updated, along with any accounts that indicate the buyer as a "parent".
  • All Related - The buyer, buyer's parent and children, and buyer's "siblings" will be updated. In other words, this includes the "Buyer and Children" as above, plus the children of the buyer's parent.

Profile Membertype
If the "Updates Profile" checkbox is selected, you can enter an optional short text field to indicate the membership type, such as "Single", "Family", "Regular", "Premium", etc. This string will be displayed in the membership lists. If nothing is entered here, the product's name will be added to the profile record.
